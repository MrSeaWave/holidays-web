# 智能休假方案计算器 UI 使用说明

## 功能简介

这是一个基于中国节假日政策的智能休假方案计算器，能够根据用户设定的参数，自动推荐最佳的休假方案。

## 主要特性

### 🎯 核心功能
- 智能计算最佳休假方案
- 支持中国节假日和调休政策
- 多种约束条件设置
- 实时方案评分和排序
- 连续休假天数统计

### 🎨 界面特色
- 现代化响应式设计
- 渐变色彩搭配
- 平滑动画效果
- 深色主题适配
- 移动端优化
- 📅 可视化日历展示
- 🎯 多方案切换对比

## 使用指南

### 基础设置
1. **日期范围**: 选择计划休假的起始和结束日期
2. **休假天数**: 设定总共需要休假的天数（1-30天）

### 约束条件（可选）
1. **不可休假日期**: 指定不能休假的特定日期
2. **强制休假区间**: 在指定时间段内必须休满一定天数

### 操作流程
1. 设置基础参数
2. 添加约束条件（可选）
3. 点击"计算最佳方案"
4. 查看推荐结果

## 结果说明

### 统计信息
- **总工作日**: 日期范围内的工作日数量
- **总节假日**: 国家法定节假日数量
- **总周末**: 周末休息日数量
- **计划休假**: 设定的休假天数

### 方案推荐
- **方案排序**: 按评分从高到低排列
- **评分机制**: 综合考虑连续性和节假日连接
- **最佳方案**: 金色标识的推荐方案
- **详细信息**: 显示具体日期、连续天数、安排描述

### 日历视图
- **可视化展示**: 直观的日历形式显示休假安排
- **颜色标识**: 不同颜色区分休假日、工作日、周末等
- **方案切换**: 支持在多个方案之间切换对比
- **日期标记**: 特殊日期显示对应的标识符号

## 评分规则

### 基础得分
- 每个休假日：+10分
- 连续休假奖励：每连续1天 +5分
- 节假日连接奖励：与节假日相邻 +15分

### 约束验证
- 严格遵守不可休假日期
- 确保强制休假区间满足要求
- 自动过滤不符合条件的方案

## 日历功能详解

### 日期颜色说明
- 🔴 **红色边框**: 计划休假日期，右上角显示"休"字标记
- 🟠 **橙色边框**: 不可休假日期，右上角显示"禁"字标记  
- 🔵 **蓝色背景**: 周末休息日
- ⚪ **灰色背景**: 普通工作日
- 🌫️ **半透明**: 超出选择范围的日期

### 交互功能
- **月份切换**: 使用日历头部的"上月"/"下月"按钮
- **方案切换**: 多个方案时可通过下拉选择器切换查看
- **悬停效果**: 鼠标悬停显示详细信息和动画效果
- **图例说明**: 日历上方提供颜色图例说明

## 注意事项

1. **日期限制**: 只能选择当前日期之后的时间范围
2. **约束冲突**: 确保约束条件之间不产生冲突
3. **性能优化**: 大范围计算可能需要较长时间
4. **浏览器兼容**: 建议使用现代浏览器获得最佳体验
5. **日历显示**: 日历仅显示基础的周末识别，不包含完整节假日信息

1. 选择日期范围：2024-01-01 至 2024-01-31
2. 设置休假天数：5天

## 常见问题

**Q: 为什么没有找到合适的方案？**
A: 可能是约束条件过于严格，尝试调整休假天数或放宽约束条件。

**Q: 如何理解评分机制？**
A: 评分越高表示方案越优，主要考虑连续性和与节假日的衔接。

**Q: 支持哪些节假日？**
A: 支持中国大陆所有法定节假日，包括春节、清明、五一、端午、中秋、国庆等。

**Q: 如何处理调休？**
A: 自动识别调休安排，周末工作日按工作日处理，平时调休按休息日处理。
